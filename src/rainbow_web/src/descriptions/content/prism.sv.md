# Vad simuleringen visar

Prismamodulen jämför luft, ett rakt block, ett vinklat block och ett triangelprisma. På så sätt blir det tydligt att dispersion kräver både olika ljushastigheter och en geometri som skickar färgerna åt olika håll.

![I ett triangelprisma bryts olika våglängder olika mycket vid varje yta.](prism-overview.svg)

# Fysikidéer

Dispersion kan skrivas som att brytningsindex beror på våglängden, alltså $n = n(\lambda)$. Kortare våglängder får ofta större brytningsindex i glas och böjs därför mer.

$$
n_{\text{violett}} > n_{\text{röd}}
$$

# Saker att prova

- Jämför först de två blocken och lägg märke till att parallella ytor inte ger någon nettospridning i riktning.
- Välj sedan prismat och se hur färgerna lämnar mediet i olika vinklar.
- Ändra infallsvinkeln och följ hur hela spektrumet vrids.
