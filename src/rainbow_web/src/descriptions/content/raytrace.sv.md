# Vad simuleringen visar

Den här modulen fokuserar på geometrin. En smal stråle träffar en sfärisk droppe och delas upp i reflekterade och transmitterade delar varje gång den möter ytan.

# Fysikidéer

Fresnels ekvationer avgör hur mycket intensitet som går till reflektion respektive transmission. I denna förenklade visualisering används de främst för att väga upp strålarnas ljusstyrka.

$$
I = I_0 R^m T^k

# Saker att prova

- Flytta källan i sidled och följ hur träffpunkten ändras.
- Jämför strålar som går nära centrum med strålar som träffar längre ut.
- Leta efter hur antalet inre reflexer påverkar den synliga intensiteten.
