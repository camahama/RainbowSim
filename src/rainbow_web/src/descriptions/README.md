# Description Files

These files are meant to be edited by non-programmers.

Use the files in `content/`:

- One file per module and language, for example `prism.sv.md` and `prism.en.md`
- Normal text becomes paragraphs
- `#` and `##` create headings
- Lines starting with `- ` create bullet lists
- Images use Markdown syntax:
  `![Caption text](/descriptions/prism-overview.svg)`
- Inline LaTeX uses `$ ... $`
- Display LaTeX uses:

```text
$$
n(\lambda)
$$
```

Image files should live in `images/` inside this descriptions folder.
